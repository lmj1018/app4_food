개인정보처리방침

메추 (Mechu) / 최종 수정일: 2026-03-01

메추(이하 "앱")는 이용자의 개인정보를 중요하게 생각하며, 관련 법령을 준수합니다. 본 방침은 앱에서 수집·이용하는 정보와 처리 목적을 안내합니다.

1. 수집하는 정보

  • 위치정보(선택): 주변 음식점 검색 기능 제공을 위해 단말의 현재 위치(위도/경도)를 사용합니다.
  • 검색어: 통합검색/추천 기능 수행을 위해 사용자가 입력한 검색어를 처리합니다.
  • 기기/앱 동작 정보(비식별): 오류 대응 및 서비스 품질 개선을 위해 최소한의 로그가 생성될 수 있습니다.

앱은 회원가입 기능을 제공하지 않으며, 이름/주민등록번호 등 민감한 식별정보를 수집하지 않습니다.

2. 정보 이용 목적

  • 주변 음식점 검색, 거리 기반 정렬, 룰렛/추천 결과 제공
  • 서비스 오류 분석 및 안정성 개선

3. 제3자 제공 및 외부 API 사용

앱은 검색 기능 제공을 위해 외부 위치/장소 API를 호출할 수 있습니다.

  • 카카오 로컬 API (장소 검색/거리 기반 검색)
  • 네이버 지도 연동 링크(사용자 선택 시 외부 지도 앱/웹으로 이동)

외부 서비스 이용 시 해당 서비스의 개인정보처리방침이 적용될 수 있습니다.

4. 보관 및 파기

  • 앱은 서비스 제공에 필요한 범위를 넘어 개인정보를 장기 보관하지 않습니다.
  • 법령상 보관 의무가 없는 정보는 목적 달성 후 지체 없이 파기합니다.

5. 권한 안내

  • 위치 권한: 주변 음식점 검색/추천 기능을 위해 사용됩니다.

권한은 단말 설정에서 언제든지 허용/거부할 수 있습니다.
