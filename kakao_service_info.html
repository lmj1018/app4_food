메추 (Mechu)

사용자 위치 기반 음식점 검색/추천 앱

메추는 현재 위치 주변의 음식점을 검색하고, 룰렛/핀볼 방식으로 랜덤 추천 결과를 제공하는 모바일 앱입니다.

주요 기능

  • 통합검색: 키워드(음식/가게명) 기반 주변 검색
  • 주변식당 기준 룰렛: 주변 음식점 후보 중 랜덤 추천
  • 음식 기준 룰렛: 음식 카테고리 기반 랜덤 추천
  • 테마추천: 미니게임(핀볼) 결과 기반 추천

카카오 API 사용 목적

본 앱은 카카오 로컬 API를 사용하여 키워드 검색/좌표 기반 검색으로 음식점 후보를 조회합니다.

  • 사용 목적: 사용자 현재 위치 기반 음식점 검색 및 추천
  • 대상 데이터: 장소명, 주소, 좌표, 카테고리 등 검색 결과 메타데이터
